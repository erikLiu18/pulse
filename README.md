# Pulse — Minimalist Time Tracker

A two-user time tracking app with 15-minute interval logging and AI-powered weekly insights.

## Local Development

```bash
npm install
npm run db:seed
npm run dev
```

Open http://localhost:5173

## Tech Stack

- **Frontend:** React + Vite + Tailwind CSS + Zustand + Recharts
- **Backend:** Express.js + TypeScript + SQLite
- **AI:** Claude API (Anthropic SDK)
- **Deploy:** Railway (Railpack)
